# Advanced Computing Research Group Website

This is the official website for the Advanced Computing Research Group (ACRG).

## Project Structure

- `index.html`: Home page containing an introduction and recent research highlights.
- `members.html`: List of faculty, students, and alumni.
- `publications.html`: List of selected publications organized by year.
- `news.html`: Archive of news and announcements.
- `style.css`: Main stylesheet.
- `script.js`: Basic interactivity (mobile navigation).

## Development

This is a static website built with vanilla HTML, CSS, and JavaScript. It is designed to be hosted on GitHub Pages.

To view the site locally, simply open `index.html` in your web browser.
